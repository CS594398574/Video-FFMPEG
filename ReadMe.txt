软件环境   Win10+VS2015+ffmpeg+opencv2.4.10

该软件需要下载ffmpeg并且写入win环境变量
本软件有切割视频、合并视频、裁剪视频、视频旋转四个功能，后续有需要在修改优化。
以下分别介绍各种功能。

切割视频参数输入：
起始时间：00:12:00 
间隔时间：00:10:00
从第12分中开始切割10分钟长的视频

合并视频
无限制打开视频将点击的视频写入F:\\Recording.txt（默认），点击清除txt，将该txt内容清空。点击合并则将该txt所有视频合并成一个视频

裁剪视频
W:裁剪的宽度
H:裁剪的高度
X:Y原视频裁剪坐标原点。即左上角位置

视频旋转
文本框默认角度：PI/2即90度旋转，可以更改参数
输出视频宽度默认：1080
输出视频高度默认：1920
旋转按钮按参数旋转。

将视频切割成帧.jpg形式存储视频目录文件下。

批量裁剪视频：点击读取视频单帧，获得视频帧，利用鼠标规划矩形框点击加入，可以多次打开读取视频，点击加入。然后批量裁剪。

综上，所有的处理视频会在原始视频的同一个目录下。
